# Pip Raine Breathwork Website

## Project Overview
Client website for Pip Raine's breathwork business. Static HTML/CSS/JS site.

## Color Palette
- Main 1: `#41178C` (deep purple)
- Main 2: `#7C178C` (purple-magenta) — alternative: `#0A1E80` (deep blue), client undecided
- Neutral: `#DFD9E2` (light lavender), `#EDF2F2` (light grey-mint)
- Button: `#7DFFE8` (bright cyan/turquoise)

## Fonts (client choosing between)
- Display: `Permanent Marker` or `Chewy`
- Body: `Ovo` or `Mate`

## Design References
- effiekli.com — menu style, welcome feel
- mindbodyhealingwithleyla.com — home page length, layout
- drjennifermullan.com — welcome-into-about-me flow
- cyndilauper.com — sharpness, colorful feel, menu, social icons top+bottom

## Key Requirements
- Home page NOT too lengthy
- Offerings at top to promote queer/community & creative/musician work separately
- About me section further down
- Welcome with audio on first view
- Fullscreen menu overlay
- Social icons in header AND footer
- "Unapologetically Being" tagline vibe
